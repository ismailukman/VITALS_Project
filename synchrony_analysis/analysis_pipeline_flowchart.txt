EGG-BRAIN SYNCHRONY ANALYSIS PIPELINE
From Raw Data to FDR-Corrected Population & Individual-Level Significance
================================================================================

This pipeline analyzes the synchronization between gastric (EGG) signals and
brain BOLD signals at both voxel and ROI levels.

SUBJECTS: VITD0107, VITD0126, VITD0128 (3 subjects with both EGG and brain data)

STEP 4: SYNCHRONY COMPUTATION (PLV & awPLV)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌─────────────────────────────────────────────────────────────────┐
    │                  PHASE LOCKING VALUE (PLV)                      │
    │                                                                 │
    │                         ┌──  T          ─┐                      │
    │                         │  ──            │                      │
    │         PLV = │ 1/T  Σ  │ e^(iΔφ(t))   │ │                     │
    │                         │               │ │                     │
    │                         └─  t=1         ─┘                      │
    │                                                                 │
    │  where:                                                         │
    │    - T = total number of time points                            │
    │    - Δφ(t) = instantaneous phase difference (BOLD - Gastric)    │
    │    - | · | = absolute value (modulus)                           │
    │    - Range: [0, 1]                                              │
    │    - 0 = no phase locking, 1 = perfect phase locking            │
    │                                                                 │
    └─────────────────────────────────────────────────────────────────┘
                                   │
                                   ▼
                        ┌────────────────────┐
                        │   PLV Empirical    │
                        │   (per subject,    │
                        │    per voxel/ROI)  │
                        └──────────┬─────────┘
                                   │
                                   │
    ┌─────────────────────────────────────────────────────────────────┐
    │          AMPLITUDE-WEIGHTED PLV (awPLV)                         │
    │                                                                 │
    │                       ┌──  T                    ─┐              │
    │                       │  ──                      │              │
    │      awPLV = │ 1/T  Σ │ w(t) · e^(iΔφ(t))      │ │             │
    │                       │                         │ │             │
    │                       └─  t=1                   ─┘              │
    │                                                                 │
    │  where the weight function is:                                  │
    │                                                                 │
    │              w(t) = A_g(t) · A_b(t)                             │
    │                     ─────────────────                           │
    │                      Σ A_g(τ) · A_b(τ)                          │
    │                      τ                                          │
    │                                                                 │
    │  where:                                                         │
    │    - A_g(t) = gastric envelope (amplitude)                      │
    │    - A_b(t) = BOLD envelope (amplitude)                         │
    │    - w(t) is normalized such that Σw(t) = 1                     │
    │    - Range: [0, 1]                                              │
    │    - Emphasizes high-amplitude coupling periods                 │
    │                                                                 │
    └─────────────────────────────────────────────────────────────────┘
                                   │
                                   ▼
                        ┌────────────────────┐
                        │  awPLV Empirical   │
                        │   (per subject,    │
                        │    per voxel/ROI)  │
                        └──────────┬─────────┘
                                   │
                                   ▼

STEP 6: INDIVIDUAL-LEVEL STATISTICAL TESTING (FIRST LEVEL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    For each subject and each voxel/ROI:

    ┌─────────────────────────────────────────────────────────────────┐
    │  Individual P-value Calculation:                                │
    │                                                                 │
    │  p_PLV = proportion of null PLV values ≥ empirical PLV          │
    │        = #{PLV_null ≥ PLV_empirical} / N_null                   │
    │                                                                 │
    │  p_awPLV = proportion of null awPLV values ≥ empirical awPLV    │
    │          = #{awPLV_null ≥ awPLV_empirical} / N_null             │
    │                                                                 │
    └─────────────────────────────────────────────────────────────────┘
                                   │
                                   ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │  Multiple Comparison Correction (Individual Level):             │
    │                                                                 │
    │  FDR Correction (Benjamini-Hochberg) across all voxels/ROIs:    │
    │                                                                 │
    │  For VOXEL analysis (~100,000 tests per subject):               │
    │    - Input: All p-values across voxels                          │
    │    - Output: q-values (FDR-corrected p-values)                  │
    │    - Threshold: q < 0.05                                        │
    │                                                                 │
    │  For ROI analysis (~132 tests per subject):                     │
    │    - Input: All p-values across ROIs                            │
    │    - Output: q-values (FDR-corrected p-values)                  │
    │    - Threshold: q < 0.05                                        │
    │                                                                 │
    └─────────────────────────────────────────────────────────────────┘
                                   │
                                   ▼
                        ┌────────────────────┐
                        │ INDIVIDUAL RESULTS │
                        │                    │
                        │ Subject-level maps │

                        │ - awPLV empirical  │

================================================================================
ANALYSIS OVERVIEW
================================================================================

DATA:
  3 subjects with both EGG and preprocessed brain (BOLD) data:
    - VITD0107
    - VITD0126
    - VITD0128

  Brain data: dswauRestingState_EGG_*.nii (4D preprocessed fMRI)
  EGG data:   gast_data_*_run1strict.npy  (preprocessed gastric signal)
  Atlas:      132-region brain atlas (Harvard-Oxford cortical/subcortical)

ANALYSIS APPROACHES:
  1. Voxel-based: Synchrony computed at every brain voxel
     - First level: Individual subject maps
     - Second level: Group statistics across subjects

  2. ROI-based: Synchrony computed per atlas region
     - Mean time series extracted per ROI
     - PLV/awPLV between gastric and each ROI
     - Significance testing with FDR correction

NOTES:
  - With only 3 subjects, group-level statistics have limited power
  - Individual-level results (first level) are more informative
  - ROI analysis provides interpretable region-specific results
  - Mismatch null distribution is limited to 2 null samples per subject

Based on analysis provided by Levakov
https://github.com/GidLev/brain_gastric_synchronization_2023/tree/master.
